# RevRent

RevRent is a modern, comprehensive vehicle rental application built with React Native and TypeScript. It provides a seamless experience for users to browse, compare, and reserve vehicles, while offering vendors a powerful dashboard to manage their fleets.

## Features

- **User Authentication:** Secure login and sign-up flows.
- **Vehicle Exploration:** Browse available vehicles and view detailed specifications.
- **Vehicle Comparison:** Compare multiple vehicles side-by-side to make an informed decision.
- **Reservation System:** Secure bookings with a token reservation hold mechanism.
- **Vendor Dashboard:** Dedicated interface for vendors to track bookings and manage inventory.
- **Feedback & Media:** Allows users to submit feedback and upload media related to their rentals.

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed along with an iOS Simulator or Android Emulator. For testing on a physical device, install the Expo Go app.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AdhyayanDubey/Revrent.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Revrent
   ```

3. Install the dependencies:
   ```bash
   npm install
   # or yarn install
   ```

### Running the App

Start the development server:
```bash
npm start
# or npx expo start
```
Follow the instructions in the terminal to open the app on your desired platform (iOS, Android, or Web).
